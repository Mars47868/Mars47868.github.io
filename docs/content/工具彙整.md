以下是我當軟體工程師以後在工作之餘所開發的一些小工具，用來解決自己或身邊親朋好友在工作或生活上的問題。
現在我想要將這些小工具進行整理，以便用於日後找新工作或者接案時，有作品集可以對外介紹說明使用。
至於作品集要用什麼形式還沒決定，但這個檔案主要先把內容撰寫出來。

1. 庫存表自動生成工具 AUI
程式位置：/Users/marschen/develop/python_workspace/AutoUpdateInventory
功能：可以透過下指令方式，快速整理每週庫存表，可用於庫存會議，整理庫存表時間從一小時縮減成5分鐘以下

程式位置中有CLAUDE.md檔，內有記錄這個工具的開發及應用紀錄，以及Usermanual檔案可供參考。
AUI 是我第一個用程式碼所編輯出來的自動化工具，因為當初我在亞果元素時，這件事情曾經是我的工作項目，
當我會編寫程式碼的時候，就想透過自動化方式來節省大量時間，而衍生出這個工具。

2. 年節訂單報表彙整
程式位置：/Users/marschen/develop/tool/好日子
功能：直接在 Excel 檔案中，透過撰寫 VBA，可以直接將 POS 機的 raw data 進行統整，並統整成餐廳人員看得懂的表單，可以快速知道年節的訂貨量。

程式位置中，可參考 好日子/報表/ 目錄，目錄底下有VBA 的原始碼檔案 ImportAndCleanPOSCsv.txt，以及提供給 user 的操作說明書。
當初會接到這個需求也是親戚在餐廳上班，每次年節前就是他的行政文書地獄，客人訂購年菜，要先將資料手寫到三聯單，隨後 key in 到 POS 機，再用三聯單逐筆 key 到excel。
原本的工作流程非常耗時、費力，直接透過 excel 巨集，利用 POS 機直接匯出的 raw data ，進行資料整理，大大節省作業時間。

3. 快速用藥查詢工具 quick-medication
程式位置：/Users/marschen/develop/tool/quick-medication
功能：透過設定常用用藥組合，可以直接透過前端介面進行查詢，方便看診時可以快速找到用藥組合

當初有個朋友在診所上班，他想要有個介面可以快速查找用藥組合，也希望這個工具可以讓原本是二線的醫生，可以在一線面對病人時，針對症狀、疾病可以快速找出用藥。
這個工具目前還沒背真的拿去使用，還算是一個模型。現有診所的醫療系統其實也都有類似的工具，可以設定快速組合的模組，但這些設定會被其他人也看到，
而這個工具當初就是要來解決這個問題，有點類似專屬的用藥組合工具。
這個工具是透過 google apps script 設計，將 raw data 存在 google 文件中，創建 apps script 來使用這個用藥查詢。

4. 股票監控台 stock_dashboard
程式位置： /Users/marschen/develop/tool/stock_dashboard
功能：可以快速掌握目前手上不同券商的股票庫存、報酬、損益等資訊。

程式位置中可以看到 CLAUDE.md、docs/spec/2026-04-21-dashboard-capital-efficiency-design.md 檔案來了解這個工具的規格及設計。
因為平常會幫家人透過自己的券商購買股票，尤其是美股，且同時會有不同券商，想要創建一個監控台，可以快速知道目前投入成本、目前持有標的、損益等等資訊，以便進行資產配置。
這個工具也是透過將交易紀錄紀錄在 google sheet，並且將程式碼寫到 google apps script中，可以部署在雲端，即時查看。
